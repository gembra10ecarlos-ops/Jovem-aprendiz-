# Quiz Jovem Aprendiz Senac

Este é um projeto interativo desenvolvido para o programa Jovem Aprendiz do Senac.

## Estrutura do Projeto

- `backend/`: Contém o servidor Express, rotas, controladores e banco de dados SQLite.
- `frontend/`: Contém a aplicação React, componentes, estilos e páginas.

## Como Executar

1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`

## Tecnologias Utilizadas

- React
- Express
- SQLite (better-sqlite3)
- Tailwind CSS
- Motion (framer-motion)
- Lucide React
